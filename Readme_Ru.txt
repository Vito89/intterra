Для демонстрации программы, достаточно, находясь в корне с проектом, в командной строке ввести строку с параметрами:

gradlew run -PappArgs="aaa bbb aaa"

Как это устроенно:
Класс на входе принимает итератор какой-либо коллекции, далее, проходит по ней наполняя карту значениями,
 где в качестве ключа используется сам элемент коллекции
 а в качестве значения - количество встретившихся элементов с аналогичным значением в коллекции.
Затем, данная коллекция упорядычивается с помощью внутреннего класса,
 с переопределенным методом "compare" и преобразуется в массив.
Массив представляет сам "топ" элементов. Он возвращается выше и будет выведен.

Возможности масштабирования:
Как вариант, можно сделать настраиваемый размер топ - коллекции, к примеру:
"new FrequencyTop(12).printTop"
при этом метод printTop сделать не статичным.
Так же имеется возможность организовать вывод в OutputStream:
"new FrequencyTop(12).printTop(elements, System.out);".
