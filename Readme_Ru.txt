Для демонстрации программы достаточно:
находясь в корне с проектом, в командной строке ввести строку на выполнение:
sh example

Как это устроенно:
В файле example (корень проекта) все необходимые команды и аргумент на вход.

Утилита игонрирует почтовые адреса не прошедшие проверку по паттерну.
При парсинге строк если обнаруживаются ошибки - строка игнорируется, обработка общего сообщения
на этом не прекращается.

Тестами покрыт как основной сервис так и Runner для контроля корректности приема сообщения извне.
